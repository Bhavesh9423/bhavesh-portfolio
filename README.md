# ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/Bhavesh9423/portfolio.git
```

### Navigate to the project

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
